Resumen de los temas que hemos visto hasta el momento

  El hardware es la parte física del computador, con esto nos referimos a todos los componentes que se pueden tocar que hacen posible que el equipo funcione, como el procesador, la memoria, el disco duro, el mouse, el monitor y el teclado. Sin estos elementos el computador no podría ejecutar ninguna tarea. Por otro lado, el software está formado por las cosas que no podemos tocar, como instrucciones, programas y sistemas que permiten que el hardware realice acciones específicas. Es lo que le da por decirlo asi vida a la máquina, permitiendo que realicemos actividades como escribir, navegar por internet, jugar o programar
  
  Los lenguajes de programación son las herramientas que usamos para comunicarnos con el computador de manera clara, ordenada y facil para los humanos. Funcionan como un puente entre las ideas humanas y el lenguaje que las máquinas pueden entender. En este caso, el lenguaje del ejemplo es Python, conocido por su sencillez y por permitir escribir código de una forma muy parecida al lenguaje natural

  Por ultimo, las variables y los valores son conceptos muy importantes. Las variables sirven para guardar información dentro del programa, como nombres, números o respuestas. Los valores son los datos que se guardan en esas variables, y pueden ser de distintos tipos: int, bool, float, string. La asignación se hace usando el símbolo '='

Ejemplo de codigo con Python

  nombre = "Ana"  #asignamos valor a nombre
  edad = 30  #asignamos valor a edad

  print(nombre, edad) #Imprime en pantalla nombre y edad
